# AI Workplace Productivity Assistant

## Project Description

The AI Workplace Productivity Assistant is a web application developed using Next.js, React, TypeScript, and Tailwind CSS. It helps users improve workplace productivity by providing smart tools for writing emails, summarizing meetings, managing tasks, generating research reports, and interacting with an AI-inspired chat assistant.

---

## Features

- 📧 Smart Email Generator
- 📝 Meeting Summarizer
- 📅 Smart Task Planner
- 🔍 Research Assistant
- 🤖 AI Chat Assistant
- ⚙️ Settings Page
- 📊 Dashboard

---

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open the application

```
http://localhost:3000
```

or

```
http://localhost:3001
```
